## Additional Backend Generator Quality Fix

The user reported two quality issues after reviewing generated MM mocks:

- Write Number From Given Position was repeatedly generating the same type of prompt, especially `Position -1`.
- Multiplication and division questions sometimes used shortcut operands such as `10`, `30`, `40`, `50`, `80`, or very low quotients, making mental calculation too easy.

Implemented:

- Competition Write Number From Given Position now uses seeded position selection instead of always inheriting the same challenge-row position.
- The MM section-locked collector avoids reusing a write-position value within the same positional section while unused positions remain.
- Whole and decimal multiplication reject trivial scale operands.
- Whole and decimal division reject trivial scale divisors and low/scale-like quotients.
