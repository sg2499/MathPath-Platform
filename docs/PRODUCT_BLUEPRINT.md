# Product Blueprint v1

MathPath Platform is a curriculum-aware dynamic MCQ practice, assessment and competition system.

## Backbone

Module → Level → Lesson → DPS → Section → Question Pattern → Generated MCQ Question → Attempt → Score/Report

## YLM Phase 1

Young Learners Module → Level 1 → Lessons 1–8 → 5 DPS per lesson → 10 MCQ questions per DPS.

## Non-Negotiables

- MCQ everywhere.
- 1 correct + 3 plausible wrong options.
- Correct option randomized.
- Numeric question labels only.
- Server-side validation, timer, and scoring.
