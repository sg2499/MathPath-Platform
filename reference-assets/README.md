# MathPath Reference Assets

This folder is for local source materials that should remain available to future Codex conversations on this machine.

Use this structure:

- `incoming/` for newly provided files before they are cataloged.
- `dps-images/` for DPS reference images.
- `workbooks/` for Excel workbooks.
- `extracted/` for local extraction outputs before they are converted into committed code/data.

Files in this folder are intentionally ignored by git except for this README and placeholder files. After adding or moving source files here, update `docs/project-memory/SOURCE_ASSETS.md`.
